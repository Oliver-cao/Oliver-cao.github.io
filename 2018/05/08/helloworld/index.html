什么是机器学习？

机器学习(machine learning)

what is machine learning？

​ 人工智能在近几年十分火热，在非计算机专业的同学眼里以为机器学习的工作是像美剧西部世界中创造机器人王国

​

​ 或者像钢铁侠的AI管家 Jarvis 那样可以管理日常生活，自主操控机甲进行战斗的人工智能管理系统。

虽然这些层次的人工智能还没实现，但是随着硬件与算法的发展。机器学习已经逐渐应用在现代生活的各个角落，如：语音助手，目标检测，自然语言处理等。如果从专业的角度来看，到底什么是机器学习呢？

机器学习：顾名思义，就是让机器进行学习，即让计算机可以达到学习某种技能的效果。在传统计算机技术中，计算机只需要让程序员输入的一段指令，让后让执行每一行指令即可。即便在图形界面下，也需要人为地点击图标，运行程序，计算结果。但是在机器学习中则不然，对计算机输入的不再是指令而是数据，让计算机通过大量的数据，这听起来不可思议，其实是可以行的通的 。看下图：

我们通过大量的数据（data）输入，对某个模型（model）进行训练得到一个相对确定的映射模型，再输入新的数据集进行预测（prediction ）得到预测结果。这样看来，机器学习的步骤是不是和人类进行学习的步骤十分类似。

通过上述流程，我们应该可以大体明白机器学习的主要流程，但是在机器学习算法中的实际设计中，涉及到了大量数学，统计学，概率论，线性代数，矩阵理论的知识。这也是要真正实现机器学习的基础。

How does machine learning work？

①选择数据：将你的数据分成三组：训练数据、验证数据和测试数据

②模型数据：使用训练数据来构建使用相关特征的模型

③验证模型：使用你的验证数据接入你的模型

④测试模型：使用你的测试数据检查被验证的模型的表现

⑤使用模型：使用完全训练好的模型在新数据上做预测

⑥调优模型：使用更多数据、不同的特征或调整过的参数来提升算法的性能表现

在上述的步骤中，每一步都涉及多种多样的算法。接下来开始深入探究机器学习。
